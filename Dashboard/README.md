# Dashboard

This folder contains the Power BI dashboard developed for the **Indian Cricket Analytics** project.

## Dashboard Pages

- Overview
- Men's Analysis
- Women's Analysis
- Player Comparison
- Statistics Table
- Player Profile (Drill Through)

## Features

- Interactive slicers
- KPI cards
- Player comparison
- Drill-through analysis
- Dynamic filtering
- Responsive dashboard layout
