# Data Merging

This folder contains the notebook responsible for merging all cleaned datasets into a single player dataset.

The merged dataset is used for PostgreSQL and Power BI reporting.

## Process

- Import cleaned CSV files
- Merge player statistics
- Combine Test, ODI and T20I records
- Prepare the final dataset for analysis
- Exported Merged CSV files

## Output

- Player_Date
- Player_Data.csv

This merged dataset serves as the primary source for the dashboard.
