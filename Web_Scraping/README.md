# Web Scraping

This folder contains the Python notebook used to extract batting statistics from the official BCCI website.

The scraping process is the **Extract** phase of the ETL pipeline.

## Data Source

Official BCCI Website

https://www.bcci.tv/international

## Extracted Data

- Men's ODI
- Men's T20I
- Men's Test
- Women's ODI
- Women's T20I
- Women's Test

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas

The extracted data is exported as CSV files for further cleaning and transformation.
